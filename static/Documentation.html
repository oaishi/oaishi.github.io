Hello!  👋

Welcome to our study! This documentation will guide you through installing and using our Chrome extension. If you have any particular questions, feel free to jump to troubleshooting at the bottom of the documentation. Leave a question or contact Faria for assistance.

Requirements

  1. Make sure to update your Chrome browser to the latest version.
  2. Browse chrome://extensions and enable ‘Developer Mode’

        

Installation

  1. Download and unzip the file on your desktop.
  2. Browse chrome://extensions and click on ‘Load unpacked’

        

  1. On the popup window, browse the folder where you unzipped the extension and click ‘Select’

        

  1. You will now see a new extension showing in the panel.

        

  1. And you are done installing it!

  1. To open the extension easily, you can also enable keyboard shortcuts by visiting ‘Keyboard shortcuts’ and setting a shortcut for activation. Here, I set it to `Ctrl+Shift+A`, which opens the extension every time I press it.

API Key setup

  1. When you open the extension for the first time, it will ask you for your OpenAI API key. Input it in the text field and press ‘Save Key.’ (We do not track your API key; it is only stored locally for action generation.) For the study, we will provide you the API Key: sk-4RscEz73zrw39ME1edhFJQ

  1. You are now ready to start using the extension!

Quick Start

  1. Please note: When you are installing the extension for the first time, the initial execution may not always work. If it does not, simply close the plugin, reload the webpage, and then reopen the plugin.
  2. Open the extension
  3. Type in your task goal and click on the start task button.
  4. The agent will now start executing the task.
  5. You can pause/stop/abort the task.
  6. Pause: To pause the agent execution and intervene, take the following steps:
  1. Pause the agent: click the pause button beside the latest action to pause the execution.

        

  1. Continue the interaction you would like to do in the website
  2. When you are ready to give the control back to the agent, click on the ‘resume’ button

  1. Successfully stop the task: By clicking on the stop button, you can mark the task as a successful task completion.

                        

        

  1. Unsuccessful/ Failure/ Abort: By clicking the abort button, you can mark the task as incomplete/unsuccessful.

  1. We recommend that you intervene at least once before marking a task as unsuccessful.
  2. If the task is doable for you to finish, we strongly recommend that you complete the rest of the actions instead of aborting.
  3. You should abort the task only when the task is impossible or you do not know how to complete the task yourself.

Limitations

  1. The agent may not always function perfectly and can be prone to errors. If an action takes too long (>30 seconds) to execute, feel free to pause/halt the execution and attempt it yourself.
  2. The agent's responses may vary across different executions.
  3. The agent might not always provide correct answers. If that happens, you are encouraged to intervene to help correct any mistakes.
  4. The agent may not work on a few websites such as Google Docs, Walmart, Chrome Settings, and Chrome extension pages.

Troubleshooting

  1. If the agent takes longer than normal to resume or to show the completion summary, this is normal; please be patient.
  2. We recommend reloading the desired webpage before you initiate a new task with the agent.
  3. Many errors stem from data processing issues and can often be resolved by rerunning the task. Reload the webpage and open the Chrome extension again if you encounter problems.
  4. If the agent stops working, go to chrome://extensions and ‘reload’ the extension to restart it.

        

  1. You may encounter a couple of warnings that are safe to ignore, such as:
  1. WebSocket connection to 'ws://localhost:3001/ws' failed

  1. Warning during compilation by Sass

If you have experience troubleshooting Chrome extensions, you can check the ‘Inspect’ panel for more details on any errors. Please don’t hesitate to reach out to Faria if you have questions or need assistance.
